About Negative Stock

Negative stock configuration determines which commodity categories can have negative inventory balances in EFRIS.

Configuration Options:
  • All Negative (101): All commodity categories allow negative stock
  • Specific Categories (102): Only specified categories allow negative stock
Use Cases:
  • Pre-selling goods before stock arrival
  • Service-based businesses with virtual inventory
  • Temporary stock discrepancies during audits
Note: This configuration is set by URA. Contact URA to modify negative stock settings.
