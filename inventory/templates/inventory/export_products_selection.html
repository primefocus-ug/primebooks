How to Transfer Products Between Companies

  1. Select your export mode and filters above
  2. Click "Export as CSV" or "Export as Excel" to download the file
  3. Login to the target company account
  4. Go to Import Products page
  5. Upload the exported file
  6. Select import mode:
    • Combined Import: If you exported "Products with Stock"
    • Products Only: If you exported "Products Only"
  7. Choose conflict resolution (Skip or Overwrite for existing SKUs)
  8. Click Import and review the results
